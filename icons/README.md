# Setup page button logos

Place these image files in this folder for the Load Accounts buttons in the setup page:

- **monarch-icon.png** — Monarch icon (used on the orange “Load/Refresh Accounts from Monarch” button).
- **projectionLab-icon.png** — ProjectionLab icon (used on the purple “Load/Refresh Accounts from ProjectionLab” button).

Recommended size: about 20×20 to 40×40 px (they are displayed at 20×20). PNG with transparency works best for Monarch; JPG is fine for ProjectionLab.
