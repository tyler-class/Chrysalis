To run a sync:
  1. Open app.monarch.com in a browser tab.
  2. Click the Chrysalis extension icon.
  3. Click Sync Now.
1 ProjectionLab Settings

Find it in ProjectionLab → Account Settings → Plugins → Enable Plugins

Opt in to ProjectionLab Early Access

If your ProjectionLab account is enrolled in Early Access, turn this on to use the ea.projectionlab.com endpoint instead of the default app.projectionlab.com.

2 Load Accounts

Load in your accounts from Monarch and ProjectionLab. After loaded, they are cached so you don't need to reload on every visit.

  • Before you click the Monarch button, make sure you have a tab open with the Monarch home page.
  • Before you click the ProjectionLab button, make sure you have a tab open with the ProjectionLab home page.
3 Map Accounts
Before you can map accounts, save your ProjectionLab API key in Step 1, then load accounts from Monarch and ProjectionLab in Step 2.

For each ProjectionLab account, select one or more Monarch accounts. Balances from multiple Monarch accounts will be summed into one ProjectionLab account. For ProjectionLab accounts that represent an asset with both a value and an associated loan, you can map separate Monarch accounts to the asset and to the loan, or choose to map only the asset side or only the loan side if you prefer.
